Digital Logic & Microprocessors

This module covers core digital logic (minimization, combinational & sequential circuits), number representation and arithmetic, and internal architecture of early x86 processors (8086, 80286) through modern concepts (MMU, paging, TSS). It also contrasts RISC and CISC philosophies.

Key Topics Explained

1. Logic Functions & Minimization: Boolean algebra, Karnaugh maps and Quine–McCluskey methods reduce expressions and gate count for efficient combinational circuit design.

2. Combinational & Sequential Circuits: Combinational circuits (adders, multiplexers) produce outputs from current inputs only; sequential circuits (flip-flops, counters) use storage elements to depend on past inputs (state).

3. Design & Synthesis: From high-level specification to gate-level implementation: truth tables → minimization → logic network → physical gates.

4. Number Representation & Arithmetic: Fixed-point and floating-point formats, two’s complement for signed integers, normalized IEEE-754 floating-point arithmetic, and common arithmetic operations (add, multiply).

5. 8086 Architecture (Intro): 16-bit microprocessor: register set (AX,BX,CX,DX, SI, DI, BP, SP), segmented memory model (CS, DS, SS, ES), flags and basic instruction categories.

6. 80286 Essentials: Extended register set and protected-mode features: descriptor tables (GDT/LDT), segmentation with descriptors, basics of paging and address translation, and page-level protections.

7. Paging, MMU & Virtual Memory: Memory Management Unit (MMU) handles logical→physical translation; paging provides page tables and protection; virtual memory allows larger address spaces via page swapping.

8. Multitasking & TSS: Task State Segment (TSS) stores task context; context switching and task scheduling enable multitasking. I/O permission bitmaps control I/O access per task.

9. Interrupts, IDT & Exception Handling: Interrupt Descriptor Table (IDT) maps interrupts/exceptions to handlers; gates and IRQ handling are central to managing asynchronous events.

10. Pentium / RISC vs CISC Concepts: High level features of Pentium-family processors (superscalar, pipelines) and the contrast between RISC (simple instructions, pipelining) and CISC (complex instructions) architectures such as SPARC vs x86.
