Graph Theory

This module explores graphs — structures made of vertices and edges — and concepts like connectivity, trees, coloring, matching, and planarity. These topics are vital in algorithms, computer networks, and optimization problems.

Key Topics Explained

1. Connectivity: A graph is connected if there is a path between every pair of vertices. A disconnected graph has multiple components.

2. Spanning Trees: A spanning tree is a subgraph that includes all vertices with minimum possible edges and no cycles.

3. Cut Vertices and Edges: A cut vertex or bridge (cut edge) increases the number of connected components when removed.

4. Covering: A vertex cover or edge cover is a set of vertices or edges touching all edges or vertices of the graph.

5. Matching: A matching is a set of edges without common vertices. A maximum matching covers the largest number of vertices.

6. Independent Sets: A set of vertices with no edges between them. Complementary to vertex cover.

7. Coloring: Assigning colors to vertices so adjacent ones have different colors. Chromatic number = minimum colors needed.

8. Planarity: A graph is planar if it can be drawn on a plane without edge crossings (e.g., K₃,₃ and K₅ are non-planar).

9. Isomorphism: Two graphs are isomorphic if their structures are identical, differing only by vertex labeling.

10. Applications: Used in network routing, circuit design, scheduling, and computational geometry.
