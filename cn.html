Computer Networks

This module covers data communication fundamentals, transmission media, network topologies, OSI & TCP/IP protocols, switching and multiplexing techniques, and key aspects of network security and administration.

Key Topics Explained

1. Channel Capacity: Defines the maximum data rate of a channel, based on Shannon’s Theorem and signal-to-noise ratio.

2. Transmission Media: Includes twisted pair, coaxial cable, fiber-optic, and wireless (radio, microwave, infrared) links.

3. Multiplexing: Combines multiple signals over one medium — types: FDM, TDM, and WDM.

4. Switching: Data forwarding using circuit switching, packet switching, or message switching.

5. ISDN and ATM: ISDN supports digital voice/data; ATM (Asynchronous Transfer Mode) uses fixed-size cells for high-speed networking.

6. LANs and Cellular Networks: LANs interconnect local devices; cellular radio supports mobile communication using frequency reuse.

7. Transmission Techniques: Analog vs Digital, asynchronous vs synchronous transmission for data communication.

8. Network Topologies: Star, Bus, Ring, and Mesh define network interconnections and reliability.

9. OSI Model: Seven-layer model — focus on Data Link, Network, and Transport layers.

10. TCP/IP, Security & Administration: TCP/IP protocols drive internet communication; firewalls, encryption, and network management ensure secure operation.
